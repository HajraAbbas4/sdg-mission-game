# SDG Mission Game Project

This project is an educational web-based game developed using HTML to promote awareness about the United Nations Sustainable Development Goals (SDGs). The game is designed for children and students to learn SDG concepts through interactive activities, quizzes, and challenges.

The project contains multiple game levels and interfaces focused on sustainability, teamwork, and problem-solving. It was developed as part of an academic and innovation-based learning activity.

## Features
- Interactive SDG-based game levels
- Educational and engaging interface
- Multiple HTML game screens
- User-friendly navigation
- Awareness about Sustainable Development Goals

## Technologies Used
- HTML
- Basic Web Design Concepts

## Applications
- Educational gaming
- SDG awareness programs
- Student learning activities
- Interactive web development practice
